# Website Log Data Processing Pipeline

This project demonstrates an end-to-end data pipeline for processing website activity logs using Python, MySQL, and Power BI.

## Project Workflow

Raw Log Data → Data Cleaning (Python) → MySQL Database → SQL Analysis → Power BI Dashboard

## Tools Used

- Python (Pandas, NumPy)
- MySQL
- SQL
- Power BI
- Jupyter Notebook

## Key Features

- Data cleaning and transformation
- Log event analysis
- User behavior insights
- Error monitoring
- Interactive dashboard
